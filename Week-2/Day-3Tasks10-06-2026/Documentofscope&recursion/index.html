Document scope accessibility describes which variables, functions, classes, or objects can be accessed from different parts of a program or document. Typical scope levels include: Global scope – accessible throughout the application or document. Module/File scope – accessible only within a specific file or module. Function scope – accessible only inside a function. Block scope – accessible only within a block such as {} in many languages.

Recursion Flow Recursion flow is the sequence of execution when a function calls itself. A recursive process typically contains: Base case – the condition that stops recursion. Recursive case – the function calls itself with modified input.
